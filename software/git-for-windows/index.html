windows 下查看端口占用的进程

netstat -ano

netstat -ano | findstr “8086” 找到PID

tasklist | findstr “PID” 查看进程信息

tasklist -fi “pid qe 1234” -v 可以列出进程的详细信息

taskkill /pid 1688 /F 杀死指定进程
